# AP2 Anestesiologia - MED 46

Site completo de revisão estratégica com:
- Abas por assunto
- Pareto Recursivo na aba Início
- Modo escuro legível
- 100 questões de treino com filtros
- Estatísticas de desempenho
- Cronograma editável
- Mapas/diagramas próprios em HTML/SVG
- Revisão final de 2h e 30min

## Como abrir
Clique em `index.html`.

## Como hospedar no GitHub Pages
1. Crie um repositório.
2. Envie `index.html`, `style.css`, `script.js` e a pasta `assets`.
3. Vá em Settings > Pages.
4. Selecione a branch `main` e a pasta `/root`.
5. Aguarde o link do GitHub Pages.

Observação: as questões são de treino no padrão das provas anexadas e focadas nos temas recorrentes.
