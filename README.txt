# Batcomputador — Projeto Redenção

Como abrir:
1. Extraia o arquivo ZIP.
2. Abra o arquivo `index.html` no navegador.

Recursos:
- HTML + CSS + JavaScript puro.
- Funciona offline.
- Salva automaticamente no navegador usando LocalStorage.
- Tem modo escuro/claro.
- Permite editar notas, objetivos, metas, riscos e próximas ações.
- Exporta/importa backup em JSON.

Arquivos:
- index.html
- style.css
- script.js
